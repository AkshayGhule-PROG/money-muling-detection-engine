TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════════════════

Issue: Frontend not loading
  Solution:
    1. Check URL: http://localhost:5173
    2. Verify npm is running in frontend directory
    3. Check browser console for errors (F12)
    4. Try hard refresh (Ctrl+Shift+R)

Issue: Backend not responding
  Solution:
    1. Check URL: http://localhost:5000
    2. Verify Flask is running in backend directory
    3. Check ports aren't blocked: netstat -ano
    4. Restart backend service

Issue: Upload fails
  Solution:
    1. Ensure CSV format is correct (comma-separated)
    2. Check file size isn't too large (recommended <50MB)
    3. Verify file has required columns
    4. Check backend logs for detailed errors

Issue: Analysis takes too long
  Solution:
    1. Normal for large networks (complex algorithms)
    2. Check system resources
    3. Try smaller dataset first
    4. Monitor network graph size

Issue: CORS errors
  Solution:
    1. Backend CORS already configured
    2. Ensure frontend runs on localhost:5173
    3. Check browser console for details
    4. Verify headers in network tab
